Logic types

In a similar fashion to variable types in programming languages, VHDL has signal types. The most common signal type in VHDL is called std_logic. This signal type can represent most of the signal states you will encounter in a circuit.

Learning about std_logic possible values

Strong low/high states

Strong high and low are the most primitive states for std_logic. They are spelled "0" and "1" in VHDL. You cannot assign to a same signal multiple strong states, as it would be similar to shorting a power supply to ground.

Weak low/high states

Unlike strong logic states, weak logic states can be assigned to the same signal as a strong state: the stronger signal will win. This can be used to set a default value to a signal.

In digital circuits they are often called "pull-up" (weak high) or "pull-down" (weak low).

High impedance

High impedance means that the signal has no value. It is often used for data busses with multiple peripherals to not drive the bus.

Don't Care

This is used as a hint for the synthesis tool so that it can better optimize the circuit.
